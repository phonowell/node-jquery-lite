# node-jquery-lite
A tiny module for Node.js to write lines in jQuery style.

## Install
```
$ npm i node-jquery-lite
```

## Methods
- `$.Callbacks()`
- `$.Deferred()`
- `$.each()`
- `$.extend()`
- `$.get()`
- `$.noop()`
- `$.now()`
- `$.param()`
- `$.post()`
- `$.serialize()`
- `$.trim()`
- `$.type()`
- `$.when()`
