# node-jquery-lite
---

A tiny plugin for Node.js to write lines in jQuery style.

## Install
---

```
npm i node-jquery-lite
```

## Methods
---

### Methods like jQuery
- `$.now()`
- `$.type()`
- `$.trim()`
- `$.param()`
- `$.extend()`
- `$.Callbacks()`
- `$.Deferred()`
- `$.get()`
- `$.post()`
- `$.noop()`

### Methods else
- `$.try()`
- `$.parseTime()`
- `$.parseString()`
- `$.parsePts()`
- `$.parseJson()`
- `$.next()`
- `$.log()`
- `$.info()`
- `$.i()`
- `$.timeStamp()`
- `$.rnd()`
- `$.timeString()`
